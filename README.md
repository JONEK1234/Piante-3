# 🌿 Botanica Evolution

Questo è il tuo diario botanico esportato e pronto all'uso su Vercel!

## 🚀 Come Caricare su Vercel in 1 Minuto

1. **Estrai questo file ZIP** sul tuo computer.
2. Inizializza un repository Git locale, aggiungilo su **GitHub** e carica i file (oppure trascinali direttamente in un nuovo repository web su GitHub).
3. Entra nella dashboard di **Vercel** e fai clic su **Add New > Project**.
4. Importa il repository creato.
5. Vercel riconoscerà automaticamente **Vite** come Framework di build.
6. Fai clic su **Deploy**! L'app sarà online in pochi secondi, collegata in tempo reale con lo stesso database Cloud configurato qui!
